# 2020-MJU-Contest-Receiver

명지대학교 공모전 우수상 탄 App의 Receiver 부분!
- Player에서 재생하는 초음파 신호를 분석
- 지금 어떤 신호를 받고 있는지 표시
  - ID = 신호
  - AMP = 신호 크기
  - Doppler Effect = 도플러 효과가 오류 수용 한계를 넘는지
    - (도플러 효과가 발생하면 초음파 주파수가 변경되므로 효과가 크게 적용되는 경우 필터링 해야 함)
